# Cafe Menu

This project is a static HTML page for Camper Cafe, established in 2020. The menu is divided into two main sections: Coffee and Desserts, each showcasing various items with 
their respective prices. The webpage includes images to visually represent the categories and uses basic HTML and CSS for structure and styling.

## Features

- **Coffee Menu:** View a list of coffee options with their respective prices.
- **Dessert Menu:** View a list of dessert options with their respective prices.
- **Visual Appeal:** Includes images for both coffee and dessert sections.

## Usage

- **Coffee Menu:** Browse through the list of available coffee flavors and their prices.
- **Dessert Menu:** Browse through the list of available desserts and their prices.

## Installation

To get started with the Camper Cafe Menu, you can simply clone the repository and open the `index.html` file in your web browser.

```bash
git clone https://github.com/florian-catalin-elisei/cafe-menu.git
```

```bash
cd cafe-menu
```

```bash
open index.html
```
